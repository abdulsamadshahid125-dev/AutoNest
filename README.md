# AutoNest — Your Complete Car Care Platform

A web-based car service platform where users can browse, book, and manage all car-related services — oil change, tire replacement, repair, detailing — from one dashboard.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS, React Router
- **Backend:** Firebase (Auth + Firestore)
- **Deployment:** Vercel
